# Docker TEE dev

Pushed to the public docker link.
[aistcpsec/tee-dev](https://hub.docker.com/repository/docker/aistcpsec/tee-dev)

These images are used mainly by [docker-taref-dev repositories](../../../../docker-taref-dev).
